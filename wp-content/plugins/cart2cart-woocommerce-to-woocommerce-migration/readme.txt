This plugin is going to move all database from your old WooCommerce store to a new WooCommerce version quickly and with no troubles.

== Description == 
Cart2Cart: WooCommerce to WooCommerce Migration plugin is the most convenient method of transferring your store data from your old WooCommerce store to a new one. You don’t have to be a professional as move is absolutely automatical and effortless. Also, your source store is not going to be downtime when the migration is in process.

= Features of automated migration =
- **Simple** - all you need is to follow Migration Wizard to proceed an automatical switch. 
- **Fast** - a couple of hours are only needed to perform migration.
- **Free Demo** - an opportunity to check the service functions and possibilities. Move up to 10 entities within 10-30 minutes.
- **Support** - Support Team is ready to answer any questions via live chat, email or by phone.

= You can migrate the following data from WooCommerce to WooCommerce =
* Products, product images, product extra fields, product attributes, product variants;
* Orders, order statuses; 
* Customers, shipping address, billing address;
* Categories, category images;
* Reviews.

*Supported WooCommerce versions:* 1.4.x, 1.5.x, 1.6.x, 2.0.x (new software versions are constantly being added).

**Please note.** *Your Source store template is not going to be moved to Target store.*

Cart2Cart: WooCommerce to WooCommerce Migration plugin perform an automated installation of Connection Bridges, providing an interaction between the stores.  Therefore, you can switch your old WooCommerce to a new WooCommerce with no stress. So, you will be able to upgrade your WooCommerce store. When plugin is installed, you will be redirected to Migration Wizard and able to start Full migration.


= Steps to take before migration =
* Install your Target WooCommerce store and make sure that both shops are online. 
* Prepare valid WooCommerce access details (host, username, password) as they will be needed for further Connection bridge installation. 


==Installation==
1. Download a zip plugin file.
1. Unarchive plugin file to your PC.
1. Upload  extracted file to plugin directory/wp-content.
1. Proceed to Admin -> Plugins, find and choose “Cart2Cart WooCommerce to WooCommerce Migration”. 
1. Insert the WooCommerce  FTP details and download a Connection Bridge. 
1. The Connection Bridge for your Target WooCommerce store will be uploaded automatically.
1. Click Start Migration button. Then you are going to be redirected to Cart2Cart site and complete Full  migration.  

== Frequently Asked Questions ==

= WooCommerce multiple languages migration =
"No other languages apart from English were migrated to WooCommerce. The products/orders/customers in other languages do not show."
It is due to the technical characteristics of the platform that multiple languages are not migrated. WooCommerce doesn't support multi-languages by default. So, Cart2Cart migrates only one default language.

= Is there a possibility to migrate reviews to WooCommerce? = 
Yes, Cart2Cart supports migration of reviews while moving to and from WooCommerce. Moreover, review ratings will be moved together with reviews.

= Can I migrate template, design and layout? =
Cart2Cart doesn’t support migration of templates, design and layouts. However, you can transfer products, customers, orders and additional options. Find more information about data you can migrate on the appropriate page from the list of Supported carts.

= Can I migrate product and category SEO URLs? =
Yes, you can migrate product and category SEO URLs, but due to technical specifications, a number of shopping carts that support this option is currently limited. WooCommerce is among those platforms, which support this option.
